About

Welcome to Tangled Bits, a small and minimal website on which I post tech news that I personally find interesting about computer systems and science. Tangled Bits is very much in beta right now, as I am experimenting with the small news format. If you find a problem with Tangled Bits, please open issue on the Github repository.

You can learn more about me here, and if you would like to get notified when I post on Tangled Bits you can subscribe to the mailing list.
